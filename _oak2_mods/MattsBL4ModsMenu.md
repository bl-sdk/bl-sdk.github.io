# Matt's BL4 Mods Menu (Unofficial)

**Matt's BL4 Mods Menu is an unofficial community-made mod.** It is not part of, endorsed by, or maintained by the PythonSDK/bl-sdk project, the official SDK mod database maintainers, Gearbox, or 2K. It is a third-party Borderlands 4 convenience UI made by Mattmab for players who already use PythonSDK mods.

The mod adds a **SDK MODS** launcher button to supported Borderlands 4 menu contexts and opens a native UMG overlay for managing installed SDK mods. From the Mods tab, you can search installed mods, filter the list, inspect selected mod metadata, preview exposed options, open supported mod settings, enable or disable toggleable mods, refresh the list, and reload compatible mods directly from the menu.

The mod also includes a Keybinds tab for viewing and rebinding SDK mod keybinds from inside the game. Keybind capture is handled through the native menu UI, including support for clearing/unbinding keys and avoiding conflicts with BL4 menu input where possible.

Quality-of-life features include adjustable text scaling, a movable launcher button, persistent settings saved under `sdk_mods/settings/MattsBL4ModsMenu/`, and a native search box for quickly finding installed mods. The launcher is designed to appear only in supported menu contexts, such as the title/main menu and pause menu, rather than appearing over inventory or other gameplay submenus.

Use this mod if you want an unofficial native Borderlands 4 UI for managing PythonSDK mods, opening mod settings, reloading compatible mods, changing SDK keybinds, searching installed mods, and adjusting the launcher position without typing console commands.
